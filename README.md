# SoftStore - E-commerce

SoftStore is [TALL Stack](https://tallstack.dev/) e-commerce project build using :

- [Laravel 12.x](https://laravel.com). 
- [Livewire v3](livewire.laravel.com).
- [FilamentPHP v3](https://filamentphp.com).
- [Alphinejs](https://alpinejs.dev/).
- [Tailwindcss](https://tailwindcss.com).
- [Vite](https://vite.dev).
- [Preline UI](https://preline.co).
- [Livewire Alert](https://github.com/jantinnerezo/livewire-alert)
- Sqlite.
- [Stripe](https://stripe.com/) (in progress)
- Stripe PHP SDK


# Features :

‚ö° Powerful & modren admin panle that build with [FilamentPHP](https://filamentphp.com).

‚ö° Markdown Editor in create products page

‚ö° Dark & Light mode

‚ö° Fast navigating using `wire:navigate` to faster page loading

‚ö° Filters & realtime search in datatables

‚ö° Real-time updates & interactive UI using [Livewire v3](livewire.laravel.com)

‚ö° Full Authentication login, registration, forgot password, reset password.

‚ö° Payment getway integration with [Stripe](https://stripe.com/)

## install & run ‚ö°

1- install laravel farmework dependencies
```
composer install
```

2- install `laravel-vite-plugin`
```
npm install
```
3- install Livewire alert 
```
composer require jantinnerezo/livewire-alert
```
4- install `tailwindcss`
```
npm install tailwindcss -D autoprefixer
&&
npx tailwind init
```

5- install `preline ui` 

```
npm i preline
```

6- run database migrations

```
php artisan migrate
```

7- run application
```
php artisan serve
```
